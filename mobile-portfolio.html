{ Availtech

Technology

  • Kendo UI
  • HTML5
  • Native iOS
  • jQuery Mobile
  • ASP.NET MVC Razor
  • Google Maps API
  • ASP.NET Web API
  • Entity Framework
  • Clickatell SMS Gateway

Avail Technologies wanted an easy way for citizens to track buses, stop locations and routes in real-time from any device. Only Falafel Software had the breadth of knowledge to deliver such a solution. On the web, Kendo UI was used in conjunction with ASP.MVC Razor to deliver a lightning fast and responsive HTML 5 site. A native iOS app was created for iPhone and iPad users, and a jQuery mobile optimized website was created for users of other devices. From any of these clients citizens can view a Google map with real time information about bus locations, bus capacities, routes, stops, delays, announcements and more; then use the advanced toolset to optimize their travel.

 
 

{ EventBoard

Technology

  • iPhone
  • Android
  • Windows 8 Metro
  • Windows Phone 7.5
  • Windows Azure
  • SQL Azure
  • OData
  • JSON
  • RadControls for Silverlight
  • RadControls for Windows Phone
  • Sitefinity 4

Falafel Software created EventBoard, the low cost, hassle free way for conference organizers to create a mobile conference program for their event. The application was built natively for iPhone, Android and Windows Phone 7. Users can quickly find interesting sessions, create an agenda, read conference related news and tweets, view the conference floor plan, and even provide session feedback electronically. The mobile applications communicate with a Windows Azure cloud based system that uses SQL Azure as the data store. A Silverlight based administration portal, which makes extensive use of Telerik’s RadControls for Silverlight, enables conference organizers to make instant changes to events, manage their data, send push notifications, and gather feedback and statistics instantly. EventBoard’s development has been featured in Telerik case study: View Here and the product has received a rave review from a world renowned event management blogger: View Here

You can learn more about the application on its Sitefinity 4 based product website: http://www.eventboardmobile.com

{ AlphaMetrix

Technology

  • ASP.NET
  • AJAX
  • RadControls for ASP.NET AJAX
  • Telerik Reporting

AlphaMetrix has created an independent platform for highly qualified investors to research, invest, and connect with hedge funds. To enhance the user’s reporting capabilities, AlphaMetrix hired Falafel Software to build a dynamic reporting system. Report definitions, formatting, conditional expressions and the user interface for report parameters are all dynamically constructed at run-time. This fully dynamic system is driven by database definitions and now reports can be built and added to AlphaMetrix’s system without new application builds.

{ Sitefinity Hub

Technology

  • Windows 8 Store App
  • RadControls for Windows 8
  • Sitefinity API
  • XAML

Falafel’s Sitefinity Hub for Windows 8 lets users of Telerik’s Sitefinity CMS system manage all of their Sitefinity sites from one Windows 8 style dashboard. The applications Live Tiles show vital site statistics on both sides of the tile and grant quick access to administration functions. The Sitefinity Hub even does something that cannot be done from any other interface: batch approvals of blog comments with just a single tap.

 
 

{ SiteBoard

Technology

  • iPhone
  • Windows Phone 7
  • REST
  • JSON
  • Sitefinity 4

SiteBoard is Falafel Software’s solution for the Sitefinity administrator on the go. Built natively for iPhone and Windows Phone 7, SiteBoard takes advantage of the Sitefinity 4 REST API to enable users to perform critical administration tasks on the go. No more rushing into the office to log out users, moderate comments, or publish pages. Now you can do it all from your mobile device.

 
 

{ SpinBoard

Technology

  • iPad

Falafel Software regularly exhibits at major tech conferences around the world. To draw attention to the booth and spice up the traditionally anti-climactic prize giveaways, Falafel developed SpinBoard, an internal prize wheel application for the iPad. Any image on the iPad can quickly be placed on the prize wheel, and the number of prize slots available can be adjusted on the fly through the admin screen. Smooth animations, a fun theme, lots of noise and great prizes encourage conference goers to step up and swipe the wheel with their finger. During the game, the prize wheel is mirrored on a large flat screen, but in times of rest, a slideshow promotes Falafel’s products and services. Find us at the next conference, and give it a try!
