{ Sitefinity - Falafel Dashboard for Sitefinity MVC

Technology

  • Kendo UI
  • HTML5
  • JavaScript
  • ASP.NET MVC Razor
  • Sitefinity 5.1

The Falafel Dashboard for Sitefinity MVC is a marketplace add-on for Sitefinity 5.1 that turns your existing dashboard into a Sitefinity command center. The dashboard takes advantage of Sitefinity 5.1's adoption of ASP.NET MVC to provide users with 10 base widgets for monitoring and administering a Sitefinity website. The widgets are all MVC based and make heavy use of Kendo UI DataViz to give administrators brilliant visual feedback. Thanks to JavaScript the dashboard is intuitively configurable, allowing users to drag widgets to their preferred locations, collapse and expand, choose widgets to display and more. The dashboard is more than just an add-on however, it also serves as a MVC based framework that allows users to create and add their own dashboard widgets.
